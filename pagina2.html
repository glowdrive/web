Productos

El producto para un control de iluminación con Arduino que se utiliza mediante una aplicación sería un
sistema de automatización que permite gestionar y controlar las luces de un hogar, oficina o cualquier
espacio desde un dispositivo móvil (teléfono o tablet), con conectividad inalámbrica (Wi-Fi o Bluetooth), que
permite a los usuarios encender, apagary regular la intensidad de las luces mediante una aplicación móvil intuitiva.
