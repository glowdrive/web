Control Inteligente de Iluminación con Arduino
Gestiona tus luces de manera eficiente desde tu aplicación móvil


Nuestra idea de proyecto se considera hacer un control de iluminación mediante una aplicación para
poder ajuastar la luz de las lampara y la intensidad, para que el cliente se sienta
conforte y comodo, junto a que no les afecte el exceso de luz.
